# Flask Docker App 🐳

A simple, dockerized Flask web app with PostgreSQL, built with best practices.

---

## 🚀 Features

- 🐍 Python + Flask app
- 🐘 PostgreSQL database
- 🐳 Docker + Docker Compose
- 🔐 Environment variable support via `.env`
- 📁 Modular structure (Blueprints, templates)
- 🌐 Bootstrap UI (optional)
